# 🎓 School Management System (Java)

This project is a console-based School Management System developed using Java.  
It allows managing different parts of a school such as students, teachers, classrooms, and facilities.

---

## 📌 Project Overview
The system simulates real-world school operations through a menu-driven console interface.  
Users can interact with different modules like student management, employees, buses, and auditorium booking.

---

## 🚀 Features

- Add and manage students (Primary & Secondary)
- Manage teachers and support staff
- Create and view classrooms
- Auditorium booking system
- Bus management
- Playground management
- Notice board system
- Interactive menu system

---

## 🧠 Concepts Used

- Object-Oriented Programming (OOP)
- Inheritance (Student → Primary / Secondary)
- Abstraction (Abstract classes like Student & Employee)
- Composition (School contains departments, classrooms, etc.)
- Aggregation (Students, buses, playgrounds)

---

## 🛠️ Technologies

- Java
- Console-based interface
- ArrayList (Collections)
